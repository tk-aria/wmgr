# 不要ファイル削除 作業報告

## 概要
プロジェクトに不要なファイル・ディレクトリを削除。

## 削除対象
| 対象 | 理由 |
|---|---|
| `old/` | 旧プロジェクト(tsrc-rs/upkg)の残骸。clap 3.x/git2 0.13等の古い依存 |
| `test_dir/` | ダミーmanifest.yml 1つだけの放置テストディレクトリ |
| `test_manifest_path.rs` | ルート直下の孤立デバッグスクリプト。`use tsrc::` と旧名を使用 |
| `manifest.yml` (ルート) | 外部リポジトリへの個人用ポインタ。プロジェクト本体ではない |

## 検証結果
- `cargo check` — OK
